
Using the  `employees`  table, get the total salary of all employees.

**Example:**

**Input:**

`employees`  table

|    Column     |  Type   |
|---------------|---------|
| id            | INTEGER |
| first_name    | VARCHAR |
| last_name     | VARCHAR |
| salary        | INTEGER |
| department_id | INTEGER |
| manager_id    | INTEGER |

**Output:**


|    Column    |  Type   |
|--------------|---------|
| total_salary | INTEGER |
